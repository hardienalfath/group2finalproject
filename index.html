What is this?

Welcome folks! This site is our main canvas for our GIS Project. So here are few key points of what we done in our project:

  • Zoomed based markers (you will know when you try it yourself ;) )
  • Customized icon styles (see attribution texts for original creator)
  • GetFeatureInfo on GlobLand30.tif file
  • Made both in Geoserver and Openlayers of VM (OSGeo) and normal OS (Win 10) (with some tweaks)
  • 3 html pages

So, please make yourself at home, and feel free to browse the site !


Moreover we are putting the script on Github if you interested!
